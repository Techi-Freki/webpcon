# webpcon
Converts webp files to either JPG or PNG.

## Usage
```
webpcon [-h] [-f {PNG,JPG}] input_file output_file

Converts webp files to either JPG or PNG.

positional arguments:
  input_file            the webp to convert.
  output_file           The file to be outputted.

options:
  -h, --help            show this help message and exit
  -f, --format {PNG,JPG}
                        The format to convert to [PNG, JPG], defaults to PNG.

```
